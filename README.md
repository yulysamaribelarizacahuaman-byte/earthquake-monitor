# 🌋 Earthquake Monitor

A real-time earthquake monitoring application built with Python and Flask.

## 🌍 Live Demo
[earthquake-monitor-tzkn.onrender.com](https://earthquake-monitor-tzkn.onrender.com)

## 📋 Description
This app fetches real earthquake data from the USGS API and displays 
seismic activity around the world, with a focus on South America.

## ✅ Features
- Fetches real-time earthquake data from USGS API
- Filters earthquakes by magnitude
- Filters earthquakes by location
- Saves data to SQLite database
- Generates earthquake statistics
- REST API endpoints

## 🛠️ Technologies Used
- Python
- Flask
- SQLite
- USGS Earthquake API
- Git & GitHub
- Render (deployment)

## 🚀 How to Run
1. Clone the repository
2. Install dependencies: pip install -r requirements.txt
3. Run the app: python app.py
4. Visit: http://localhost:8080
